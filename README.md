# Research Notes

A scratchpad for research findings and web research outcomes.

Visit the GitHub Pages site: https://ottokruse.github.io/research-notes/

## Organization

- Files are organized with flat structure using dated filenames
- Format: `YYYY-MM-DD-topic-name.md`
- All research notes are Markdown files

## Usage

This repository is primarily written to by the Web Research Agent for storing research findings.
